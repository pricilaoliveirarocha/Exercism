# 🧠 Exercism Exercises

This repository contains my solutions to exercises from **[Exercism](https://exercism.org/)**.

Exercism is a platform designed to improve programming skills through practical exercises and mentoring.  
The purpose of this repository is to document my learning journey, organize my solutions, and track my progress over time.

---

## 📂 Repository Structure

Exercises are organized by **programming language** and **exercise name**.

Example:

├── solutions/ <br>
│ ├── language/ <br>
│ │ ├── exercise-name/ <br>
│ │ │ ├── 1/ <br>
│ │ │ ├── 2/ <br>
│ │ │ ├── 3/ <br>
│ │ │ └── ... <br>
│ <br>
└── README.md <br>

Each exercise folder may contain:

- 💻 The exercise solution  
- 🧪 Test files provided by Exercism  
- 📄 Supporting files required to run the exercise
